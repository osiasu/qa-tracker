Cloud sync — Supabase (free)
Store test cases in your own Supabase project so the whole team shares the same data in real time.
1Sign up free at supabase.com → create a project.
2Open the SQL Editor and run the setup query below once to create the table.
3Go to Project Settings → API → copy the Project URL, anon key, and service_role key.
4Paste below and click Connect. Share the Project URL + anon key with viewers.
▶ Show one-time SQL setup 
create table if not exists qa_tracker_data (
  id integer primary key,
  cases jsonb not null default '[]',
  updated_at timestamptz default now()
);
insert into qa_tracker_data (id, cases)
  values (1, '[]') on conflict (id) do nothing;
alter table qa_tracker_data enable row level security;
create policy "public_read" on qa_tracker_data
  for select using (true);
